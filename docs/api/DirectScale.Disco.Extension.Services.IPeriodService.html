System.DateTime date

Returns
Type Description
Period

The period returned will contain information about the time period and the template being used, but the profile has not been committed, so no results information will be populated.
Remarks

This method may be called to get some information about committed periods as well, such as the begin and end dates, but the result information will not be included.
