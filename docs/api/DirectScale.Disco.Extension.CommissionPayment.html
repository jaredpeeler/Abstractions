TransactionNumber

The transaction number tracking this payment.

Declaration
public string TransactionNumber { get; set; }
Property Value
Type Description
System.String
Remarks

This is usually the 3rd party merchant's transaction number. This identifier links DirectScale's record of the transaction to the transaction information available from the external payment processing entity.

When passed into PayCommissions(Int32, List<CommissionPayment>) it will usually be empty the first time a payment is attempted, but will contain the previous attempt's transaction number if it is attempting to re-process payment.
