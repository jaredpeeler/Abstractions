    Class ApiEndpoint<TRequest>

    A simplified interface to implement to create a custom API, compared to IApiEndpoint. A model will be defined as a generic TRequest as part of your custom API. So that when a request is received for your endpoint, the body of the request can automatically be parsed out into the specified model. Then the custom API code will be called with just that model directly, and not have to work with any of the http request information directly.

    Remarks

    TRequest will be created by reading JSON from the body of the request. Make sure that TRequest can be created from a JSON string.

    This interface will not give you any headers or query string arguments unless Post(ApiRequest) is overridden or IApiEndpoint is implemented directly.

    Currently only custom apis receiving POST requests are supported. Custom API GET requests are unsupported.
